# About

This dashboard shows:
- Units sold trend
- Profit by region, state, and city
- Operating profit breakdown
- Top states and cities performance

You can open the data file in Excel, Power BI, Tableau, or any spreadsheet tool.

# How to Use

1. Download the data file from the repository
2. Load it into your tool of choice (Power BI / Excel / Tableau)
3. Recreate or explore your own visualizations
